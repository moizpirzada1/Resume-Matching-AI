# 📄 AI Resume Screening System (ATS)

---

## 🧠 Overview

This is an **AI-powered Applicant Tracking System (ATS)** that automatically screens and ranks multiple resumes based on their relevance to a given job description.

It uses **Natural Language Processing (NLP)** techniques to compare resumes with job descriptions and provides a similarity score, match level, and keyword overlap.

---

## ✨ Features

- 📂 Upload multiple resumes (PDF)
- 🧠 NLP-based text extraction and preprocessing
- 📊 Resume ranking using TF-IDF + Cosine Similarity
- 🔑 Keyword matching between resume and job description
- 🏆 Highlights top candidate automatically
- 📈 Progress bar visualization for each candidate
- 📥 Download results as CSV file
- 💼 Clean Streamlit UI for easy use

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
- Pandas 📊
- Scikit-learn 🤖
- NLTK 🧠
- PyPDF2 📄

---

## ⚙️ How It Works

1. Upload multiple resume PDFs
2. Paste job description
3. System extracts text from resumes
4. Text is cleaned and preprocessed
5. TF-IDF vectorization is applied
6. Cosine similarity calculates match score
7. Candidates are ranked automatically

---

## 📊 Output Includes

- Candidate name
- Match score (%)
- Match level (Strong / Moderate / Weak)
- Common keywords
- Ranked leaderboard
- Downloadable CSV report
